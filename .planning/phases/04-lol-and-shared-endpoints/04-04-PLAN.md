Create 2 Regional-routed tournament endpoint modules. These are unique because they include POST endpoints with request bodies.

**tournament-v5.ts** -- `export const tournamentV5`:
Imports: `WhisperClient` from `'../core/client.js'`, `RegionalRoute` from `'../types/regional.js'`, all needed types from `'../types/overrides/lol-tournament.js'`.

Methods:
1. `createProvider(client, route, body: ProviderRegistrationParameters)` -> `Promise<number>` -- POST to `/lol/tournament/v5/providers`, methodId: `tournament-v5.createProvider`. Call `client.request<number>(route, path, methodId, { method: 'POST', body: JSON.stringify(body) })`. Returns provider ID (number).
2. `createTournament(client, route, body: TournamentRegistrationParameters)` -> `Promise<number>` -- POST to `/lol/tournament/v5/tournaments`, methodId: `tournament-v5.createTournament`. Returns tournament ID (number).
3. `createTournamentCode(client, route, tournamentId: number, body: TournamentCodeParameters, options?: { count?: number })` -> `Promise<string[]>` -- POST to `/lol/tournament/v5/codes`, methodId: `tournament-v5.createTournamentCode`. Pass `tournamentId` and optional `count` as query params. Body is the TournamentCodeParameters. Returns array of tournament code strings.
4. `getTournamentCode(client, route, tournamentCode: string)` -> `Promise<TournamentCodeV5>` -- GET `/lol/tournament/v5/codes/${tournamentCode}`, methodId: `tournament-v5.getTournamentCode`.
5. `updateTournamentCode(client, route, tournamentCode: string, body: TournamentCodeUpdateParameters)` -> `Promise<{ status: number }>` -- PUT to `/lol/tournament/v5/codes/${tournamentCode}`, methodId: `tournament-v5.updateTournamentCode`. Returns 204 no-body response. Implement as: call request, return `{ status: response.status }`.
6. `getLobbyEventsByCode(client, route, tournamentCode: string)` -> `Promise<LobbyEventV5Wrapper>` -- GET `/lol/tournament/v5/lobby-events/by-code/${tournamentCode}`, methodId: `tournament-v5.getLobbyEventsByCode`.

Note: Verify exact paths from Riot API explorer using Playwright. The paths above follow Riot's V5 tournament API convention.
