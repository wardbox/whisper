<threat_model>

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| repo→npm registry | LICENSE/README content becomes legally binding artifact distributed to consumers worldwide. Bad license text = legal exposure. |
| repo→npm package metadata | `homepage`, `repository`, `bugs` URLs are clickable from npmjs.com. Wrong URL = phishing vector if attacker controls the wrong domain. |
