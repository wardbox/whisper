**MemoryCache class implementing CacheAdapter:**
- Internal store: `Map<string, { value: unknown; expiresAt: number }>`
- `get<T>(key)`: Check if entry exists and `Date.now() < expiresAt`. If expired, delete it (lazy eviction) and return `undefined`. If TTL was 0, it should not have been stored.
- `set<T>(key, value, ttlSeconds)`: If `ttlSeconds <= 0`, do NOT store (TTL 0 means no caching). Otherwise store with `expiresAt = Date.now() + ttlSeconds * 1000`.
- `delete(key)`: Remove from map.
- `has(key)`: Check existence AND expiry. Delete if expired, return false.

**resolveTtl function:**
```typescript
export function resolveTtl(path: string, config: CacheTtlConfig): number {
  for (const [pattern, ttl] of Object.entries(config)) {
    if (pattern === 'default') continue;
    if (path.includes(pattern)) return ttl;
  }
  return config.default;
}
```

**buildCacheKey function:**
```typescript
export function buildCacheKey(apiKey: string, route: string, path: string, params?: Record<string, string>): string {
  const keyPrefix = simpleHash(apiKey).slice(0, 8);
  const paramStr = params ? '?' + new URLSearchParams(params).toString() : '';
  return `${keyPrefix}:${route}:${path}${paramStr}`;
}

function simpleHash(str: string): string {
  let hash = 5381;
  for (let i = 0; i < str.length; i++) {
    hash = ((hash << 5) + hash + str.charCodeAt(i)) | 0;
  }
  return Math.abs(hash).toString(36);
}
```
