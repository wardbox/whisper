**normalizeKeyProvider function:**
Returns an object `{ getKey(): Promise<string>, invalidate(): void }` that:
- For string input: always returns the string (no caching logic needed)
- For async function input: caches the resolved value. `invalidate()` clears the cached value.
- Race condition prevention: if `invalidate()` is called while a fetch is in-flight, subsequent `getKey()` calls share the SAME pending promise. Implementation: store a `pending: Promise<string> | null` field. On `getKey()`, if pending exists, return it. If cached exists, return it. Otherwise, set `pending = fn()`, await, store in cache, clear pending.

```typescript
export interface KeyProvider {
  getKey(): Promise<string>;
  invalidate(): void;
}

export function normalizeKeyProvider(input: ApiKeyProvider): KeyProvider
```

Create `packages/whisper/src/core/client.ts`:

**WhisperClient interface (the return type of createClient):**
```typescript
export interface WhisperClient {
  /** Make an API request with full pipeline: middleware -> cache check -> rate limit -> fetch -> cache store -> middleware response */
  request<T>(route: PlatformRoute | RegionalRoute, path: string, methodId: string, options?: { method?: string; body?: string; headers?: Record<string, string>; params?: Record<string, string> }): Promise<ApiResponse<T>>;
}
```

**createClient function:**
```typescript
export function createClient(config: ClientConfig): WhisperClient
```

Implementation:
1. Normalize key provider: `const keyProvider = normalizeKeyProvider(config.apiKey)`
2. Create HTTP client: `const http = createHttpClient(keyProvider)`
3. Create rate limiter (if not false): `const rateLimiter = config.rateLimiter !== false ? new RateLimiter(typeof config.rateLimiter === 'object' ? config.rateLimiter : undefined) : null`
4. Create cache (if not false): Use `config.cache` if provided and not false, otherwise create `new MemoryCache()`. If `config.cache === false`, set to null.
5. Get middleware array: `const middleware = config.middleware ?? []`
6. Get cache TTL config: `const cacheTtl = config.cacheTtl ?? { default: 300 }`

**request() pipeline:**
```
1. Build cache key (if cache enabled): buildCacheKey(await keyProvider.getKey(), route, path, params)
2. Check cache: if hit, return cached value (skip rate limiter and fetch)
3. Wrap the actual fetch in middleware pipeline:
   a. Build RequestContext { url: buildUrl(route, path), method, headers, body, route, methodId }
   b. executePipeline(middleware, context, async (ctx) => {
        - If rate limiter: await rateLimiter.acquire(route, methodId)
        - try { const response = await http.request(ctx.url, ...) }
        - If rate limiter: rateLimiter.update(route, methodId, response.headers)
        - return response
      })
   c. On RateLimitError: if rate limiter, await rateLimiter.handle429(route, methodId, error headers, attempt), retry up to 3 times
4. Store in cache with resolved TTL (if cache enabled and TTL > 0)
5. Return response
```
