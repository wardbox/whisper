Whisper is a TypeScript library that wraps every Riot Games API endpoint with full type safety, proactive rate limiting, and tree-shakeable per-game imports.

## Features

- **Zero runtime dependencies** -- uses native `fetch` only
- **Proactive rate limiting** -- prevents 429s before they happen
- **Tree-shakeable** -- import only the game you need
- **Type-safe routing** -- platform vs regional enforced at compile time
